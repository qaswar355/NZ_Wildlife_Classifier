# NZ Wildlife Classifier

An end-to-end wildlife camera trap classifier for New Zealand conservation.

## Overview

Binary image classification system that distinguishes **mammals** (invasive predators — positive class) from **birds** (native species — negative class) using camera trap imagery. Built as part of a Masters of AI thesis at Victoria University of Wellington (Te Herenga Waka).

**Dataset**: Trail Camera Images of New Zealand Animals (LILA Science) — 4,648 images, 40+ species.

## Goal

Support conservation practitioners by automatically flagging invasive predator detections, minimising false positives (a bird misclassified as a mammal could trigger harmful intervention).

## Project Status

Planning phase — implementation begins June 2026.
